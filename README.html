Tables

The database contains the following tables:
Table Name Column Description Type
users id Identifying key for students INTEGER
first_name User's first name TEXT
last_name User's last name TEXT
privilage The permissions a user will have (Peer Mentor, Admin, Teacher, Student) VARCHAR(12)
user_info
user_id References users(id) INTEGER
email User's email address VARCHAR(255)
passwd User's password VARCHAR(255)
courses
id Same as Longwood's CRN. Indentifier for courses INTEGER
semester The semester the course was offered VARCHAR(3)
year The year the course was offered INTEGER
department The course's subject or department (i.e LSEM, CMSC) VARCHAR(5)
description A written description of the course TEXT
class_key A key to be distributed by instructors so that students may be added to a course. VARCHAR(12)
course_rosters
course_id Reference to courses(id) INTEGER
user_id Reference to user(id) INTEGER
